# Tableau Dashboard Project

## Project Overview
This project involves building an interactive Tableau dashboard to analyze and visualize data insights effectively.  
The dashboard helps users understand key trends, patterns, and performance metrics through clear and intuitive visualizations.

---

## Objective
- To analyze the given dataset and extract meaningful insights  
- To design an interactive and user-friendly Tableau dashboard  
- To present data-driven findings in a visual and understandable format  

---

## Tools & Technologies Used
- Tableau Desktop  
- Tableau Public  
- Microsoft Excel / CSV (for data preprocessing)  

---

## Dataset
- Source: https://www.kaggle.com/datasets/prevek18/ames-housing-dataset
- Format: CSV / Excel  


---

## Key Insights / KPIs

- Trend of Sale Prices Over the Years

- Average Sale Price by Neighborhood

- Relationship Between Living Area and Sale Price

- Impact of Number of Bedrooms on Sale Price
